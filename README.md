# 📊 Sales Analytics Project (Google Sheets)

## 🔍 Project Overview
This project demonstrates an **end-to-end sales analysis workflow** using Google Sheets.  
It covers **raw data collection, cleaning, transformation, analysis, and dashboard visualization**.  
The live dashboard allows anyone to explore and interact with the data.

---

## ⚙️ Tools & Technologies
- Google Sheets (Data management, analysis, visualization)
- Google Apps Script (Optional, can be added later for automation)

---

## 🚀 Workflow
1. **Raw Data Collection:** Imported real-world sales data into Google Sheets.  
2. **Data Cleaning & Preparation:** Removed duplicates, formatted dates, and standardized fields.  
3. **Final Data Transformation:** Created calculated fields like Total Sales, Revenue per Product, etc.  
4. **Data Analysis:** Generated KPIs and insights to identify trends and patterns.  
5. **Dashboard Visualization:** Built an interactive dashboard for easy data exploration.
